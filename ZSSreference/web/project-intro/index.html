端到端业务流程

一句话的完整生命周期，四个阶段：

🔄 端到端业务流程图 —— 19 步时序：信令握手 → VAD 切句 → ASR → TMT 快翻 → LLM 纠错 → TTS 配音

① 信令握手

浏览器 → WebSocket → api-server → etcd 选节点 → SDP/ICE 转发 → 浏览器与 PBX 直达 WebRTC

② VAD 切句 + 流式 ASR

Opus → PCM16/16k → Silero ONNX 推理切句 → 开 ASR WebSocket → partial 实时回推（灰字）→ VAD 静音关句 → final 定稿（黑字）

③ 两段翻译（核心卖点）

TMT 快翻 < 500ms 秒出灰色草稿 → DeepSeek Flash 带上下文+术语表 polish → 跟草稿 diff → 不同则黑字+高亮修正

④ TTS 配音

api-server 发 tts_command → PBX 合成 → PCMU 编码 → 20ms 帧节拍下行播放，与字幕解耦
